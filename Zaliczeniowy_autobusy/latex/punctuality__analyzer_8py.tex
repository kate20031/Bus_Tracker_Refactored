\doxysubsection{Detailed Description}
Module for analyzing punctuality of buses by comparing real-\/time data with the official timetable. 

This module defines a class {\ttfamily Punctuality\+Analyzer} to evaluate how closely real bus arrivals match their scheduled times. It also provides a utility function to save the analysis results.

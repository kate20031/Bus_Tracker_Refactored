\doxysubsection{Detailed Description}
Module for loading bus tracking data and bus station coordinates. 

This module contains functions for\+:
\begin{DoxyItemize}
\item Fetching live bus tracking data and saving it to files depending on the hour.
\item Retrieving bus station coordinates.
\item Loading timetable data for specific bus lines and stops.
\item Reading processed data from CSV files.
\end{DoxyItemize}

It relies on Warsaw\textquotesingle{}s public API and custom processing utilities.

to fetch real-time bus location data. Based on the time of day, it stores

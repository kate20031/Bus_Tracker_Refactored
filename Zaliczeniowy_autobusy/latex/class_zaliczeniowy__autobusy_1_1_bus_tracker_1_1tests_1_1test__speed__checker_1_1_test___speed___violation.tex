\doxysubsection{Detailed Description}
Unit test case for speed violation checking functions. 

This test case includes tests for detecting speed violations in bus data, including handling invalid coordinates and multiple violations in the data. It uses a {\ttfamily \doxylink{class_zaliczeniowy__autobusy_1_1_bus_tracker_1_1data__processing_1_1speed__violation__checker_1_1_speed_violation_checker}{Speed\+Violation\+Checker}} and {\ttfamily \doxylink{class_zaliczeniowy__autobusy_1_1_bus_tracker_1_1data__processing_1_1speed__calculation__strategyd1e0db075462cb52c7c32fddc0b3ff32}{Haversine\+Speed\+Calculation\+Strategy}} to process and check for violations based on GPS data.

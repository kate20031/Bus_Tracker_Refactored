\doxysubsection{Detailed Description}
Module for analyzing bus movement data and identifying violations. 

This module provides functions for analyzing public bus data\+:
\begin{DoxyItemize}
\item counting buses exceeding speed limits
\item identifying violation hotspots (based on geographic proximity)
\item reporting punctuality issues
\end{DoxyItemize}

Data is loaded from preprocessed files representing rush and non-\/rush hour timeframes.

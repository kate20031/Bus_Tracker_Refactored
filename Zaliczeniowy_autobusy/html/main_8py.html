Detailed Description

This module runs the bus analytics process, including loading data and performing the data analysis.

This module contains functions to load bus tracking data, bus station coordinates, and to perform the analytics including vehicle speed violations, violations in vicinity, and punctuality accuracy.

Zaliczeniowy_autobusy.BusTracker.main.analyse_data ( )

Analyzes the loaded bus data.

This function calls various analytics functions to assess:

  • Vehicles that exceed the speed limit (print_vehicles_over_speed_limit()).
  • Violations occurring in the vicinity of other violations (print_violations_in_vicinity()).
  • Punctuality of buses based on the loaded timetable and actual arrival times (print_punctuality_accuracy()).
Calls functions to print vehicles over
 speed limit and violations in vicinity.

Zaliczeniowy_autobusy.BusTracker.main.load_data ( )

Loads bus tracking data and bus stations coordinates.

This function calls two separate functions to load bus tracking data and bus station coordinates. It calls load_bus_tracking_data() to fetch real-time bus location data and load_bus_stations_coord() to fetch coordinates of bus stations.

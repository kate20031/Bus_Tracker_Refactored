\doxysubsection{Detailed Description}
This module runs the bus analytics process, including loading data and performing the data analysis. 

This module contains functions to load bus tracking data, bus station coordinates, and to perform the analytics including vehicle speed violations, violations in vicinity, and punctuality accuracy.

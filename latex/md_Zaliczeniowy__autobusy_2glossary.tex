У цьому проєкті кожен автобус має\+:
\begin{DoxyItemize}
\item унікальний ідентифікатор;
\item розклад руху;
\item GPS-\/дані для відстеження позиції;
\item список зупинок маршруту.
\end{DoxyItemize}\hypertarget{md_Zaliczeniowy__autobusy_2glossary_autotoc_md3}{}\doxysubsection{\texorpdfstring{Violation (Порушення)}{Violation (Порушення)}}\label{md_Zaliczeniowy__autobusy_2glossary_autotoc_md3}
У цьому проєкті violation для автобуса — це\+:
\begin{DoxyItemize}
\item перевищення швидкості;
\item відхилення від розкладу (прибуття раніше/пізніше).
\end{DoxyItemize}

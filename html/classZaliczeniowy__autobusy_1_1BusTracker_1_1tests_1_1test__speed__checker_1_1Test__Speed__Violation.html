Detailed Description

Unit test case for speed violation checking functions.

This test case includes tests for detecting speed violations in bus data, including handling invalid coordinates and multiple violations in the data. It uses a SpeedViolationChecker and HaversineSpeedCalculationStrategy to process and check for violations based on GPS data.

Test cases for functions in the bus_data_calculations module.

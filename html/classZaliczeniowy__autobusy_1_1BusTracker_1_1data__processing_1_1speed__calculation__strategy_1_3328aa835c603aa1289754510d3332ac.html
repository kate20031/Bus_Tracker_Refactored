Detailed Description

Concrete implementation of SpeedCalculationStrategy using the Haversine formula.

Calculates distance between two geographic coordinates and converts the result into speed using a time delta between two records.

Calculates speed using the Haversine formula.

Computes distance between two consecutive points and divides by the time difference, applying a conversion factor to obtain the speed in desired units (e.g., km/h).

Parameters
